## RootPanel
A pluggable Linux virtual host selling system.

一个插件化的 Linux 虚拟主机销售系统。

RootPanel 可以用来管理和销售 Linux 服务器上的各种服务，每种服务(如 SSH, MySQL, ShadowSocks)均通过一个插件来支持。

## 核心功能

* 用户系统
* 工单系统

## 支持的服务

* MySQL
* MongoDB
* SSH
* Nginx
* PHP-FPM
* ShadowSocks
* PPTP VPN

## 插件支持的更多功能

* 用户手册
* 用户剩余时长计时
* Bitcoin 支付

## 运行环境

* PHP5.4 + MySQL, Memcache
* Debian, Archlinux, Ubuntu, Centos
